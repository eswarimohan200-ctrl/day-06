# ATS Resume Optimizer

A professional ATS Resume Optimizer prompt for ChatGPT, Claude, and Gemini.

## Features

* ATS-friendly resume optimization
* Recruiter-readable formatting
* One-page A4 resume generation
* Job description keyword alignment
* Truthful content enhancement
* ATS score estimation

## Usage

1. Copy the prompt.
2. Paste your resume.
3. Optionally add a job description.
4. Get an ATS-optimized resume instantly.

## Output

* ATS Score Analysis
* Optimized One-Page Resume
